# 강제 규율 4부 골격 (Discipline Skeleton)

> **정본**: `CLAUDE.md` §"강제 규율 4부 골격 (Discipline Skeleton)" 🔒 LOCKED 2026-06-16. 본 노트는 그 헌법 §의 *그래프 노드 포인터*.

## 요지

*꼭 지켜야 하는데 자꾸 새는* 규율(게이트·하네스·디스패치 규칙·discipline 스킬)은 서술형이 아니라 **우회 봉쇄 4부 골격**으로 쓴다:

1. **철칙(Iron Law)** — 비협상 1문장, "X 없이 Y 없음" 꼴, 볼드. 규율당 정확히 1개.
2. **단계(Phases)** — 강제 순서 + 각 단계 *완료 판정* 명시.
3. **금지 합리화 표 + 적신호(red-flags)** — 회피 사고 실명 나열·봉쇄. 끝에 "letter 위반 = spirit 위반" 박제.
4. **예외 + 탈출구** — 적용범위 + 면제 조건.

적용: discipline 규율(✅) / 기존 규율=점진 backfill(🟡) / 정의·서사형 학습=억지 적용 X(❌).
