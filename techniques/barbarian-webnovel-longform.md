# 게임 속 바바리안으로 살아남기 — 장편 문법 해부 v1 (구조 스캔)

> 용도 = 파이프라인 계약 ⑤(캠페인/장편 문법)의 웹소설 트랙. 학습 규약 동일: 원문 학습 자유·산출 복제 금지(n-gram 게이트). **소재 정합 보너스**: 주인공이 "NPC 동료 필수·사망 시 초기화·고자유도 싱글 RPG"를 공략하는 겜판 — 플레이어 심리·게임 시스템 서사화의 이중 레퍼런스.

## 1. 구조 실측 (전권 스캔)

| 축 | 값 |
|---|---|
| 총량 | 770화 · 526만 자 · 12.1MB |
| 화당 분량 | **중앙값 6,666자**(최소 2.5천 · 최대 1.6만) — 웹연재 표준(5~5.5천)보다 약간 김 |
| 아크 구조 | 연작 소제목 "(1)(2)(3)" 단위 — **아크 175개 · 중앙값 3~4화**(미니 아크 연쇄) |
| 아크 예 | 튜토리얼(3) → 기브 앤 테이크(3) → 밤친구(3) → 2층(3) → 귀환(3) → 라프도니아(4)… |

**장편 문법의 골격 = 3~4화 미니 아크의 연쇄**: 하나의 문제 제기→해결이 3~4화(2~2.7만 자)에 닫히고, 아크가 층(지역) 진행이라는 대구조에 꿰임 — 캠페인의 "시나리오 연쇄 + 대아크"와 동형. 우리 매핑: **미니 아크 ≈ 시나리오 팩 1개, 층 진행 ≈ 캠페인/시즌.**

## 2. 화 말미 절단 문법 (클리프행어 5형 — 무작위 8화 표본에서)

> ✅ **교차 검증(2026-07-18)**: [webnovel-writer 해체 — 장편 서사 시스템의 골수 6종](webnovel-writer-teardown.md)의 추독력 taxonomy(훅 5형: 위기·수수께끼·갈망·정서·선택)와 독립 수렴 — 본 실측 5형과 사실상 동일 분류. 운용 규칙(3연속 금지·1요소 차별화·미세 회수)은 그쪽이 정본.

1. **다음 위협 예고**: "아무래도 (그것이) 근처에 있는 것 같다."
2. **시스템 창 정산 마감**: 「종합 전투 지수: 982.55 (+136.25) / 획득 정수: …」 — **화가 상태창 로그로 끝남**(겜판 특유: 정산이 보상 카타르시스이자 다음 화 훅).
3. **시간 도약 훅**: "시간이 흘러 경매일이 밝았다."
4. **정체 공개 직전 절단**: "요, 용살자, 리갈 바고스…." (호명 후 컷)
5. **정보 떡밥**: "오직 9층에서만 출현한다." (규칙 한 줄 던지고 컷)

→ 우리 반영처: ①씬 말미 훅 저작 규칙(조우 카드 결과 서술의 마지막 문장 유형 5형) ②**시스템 창 정산 = 우리 시스템 라인·run 요약과 동일 문법** — 겜판 독자가 이미 훈련된 리듬이라 텍스트 RPG UI와 궁합 검증됨 ③신화 예고(소문 툴팁)의 문형.

## 3. 표본 정독 소견 (1화)

- 문체: 짧은 단문 연타 + 구어 냉소("망겜이라 하는 그들보단") + 의성어 라인 독립("타다닷, 딸깍.") — 리듬이 UI 로그와 유사.
- 도입 설계: 병원→게임 인생→[던전 앤 스톤] 발굴→2년 공략 실패→(빙의 직전) — 1화 안에 "이 게임이 왜 특별한가"의 시스템 프레젠테이션이 들어감(세계관 설명을 게임 리뷰 화법으로 처리 — 설정 덤프 회피 기술).

## 3.5 삼축 문법 (★작가 지정 핵심 — "생태계·상호작용·밸런스/성장을 내 것으로" · 2~5화 정독 추출)

### 축 1 — 생생한 게임 생태계
- **편의의 박탈이 생생함을 만든다**: 게임에서 "제작자가 플레이어에게 준 편의"(안전한 스폰·광원·상태창 호출)가 이 세계에선 제거됨 — 암흑지대 스폰·상태창 호출 실패. 세계가 진짜인 이유 = **UI가 나를 위해 존재하지 않음**. (우리 번역: 정보 비대칭 설계 — 시스템 라인은 정직하되 세계는 플레이어 편의로 굴러가지 않는다.)
- **몬스터 = 도감 스탯이 아니라 행동 생태**: "최약체" 고블린이 덫을 놓고 기다리고, 죽은 척을 의심하고, 돌을 던져 확인한 뒤 접근 — 위협은 수치가 아니라 **행동 패턴의 영악함**에서 나옴. → 우리 위협 조우·괴이 저작의 제1규칙 후보: "괴이마다 사냥 절차를 저작하라(스탯보다 먼저)."
- **사회 규칙의 자율 작동**: 성인식·악령 즉결 처형(**메타 발화 = 세계 내 사형 사유** — 게임 지식 발설이 위험이라는 장치), 바바리안 고용 차별·세금 사형 제도(직업 생태의 벽), 야만인 리더 교체 소동(사회가 코미디와 논리로 굴러감). 세계 설명이 아니라 **세계가 주인공에게 집행됨**.
- **시스템의 소름 장치**: 상태창 로그가 지구의 빈 방 컴퓨터에 실시간 기록됨 — 시스템 존재 자체가 미스터리 떡밥.

### 축 2 — 인물 상호작용
- **한 문장으로 스테레오타입 뒤집기**: 아이나르 — 서툴고 딱딱한 종족 말투를 유지한 채 "죽는 건 당연히 두렵다. 내색은 않겠지만" 한 줄로 야만인 전체가 입체화. **종족 문법(말투) 유지 + 내면 한 줄 반전** = NPC 입체화의 최소 비용 공식.
- **관계의 화폐**: 작은 도움(길 안내) → 보답 약속("살아 돌아온다면 보답하겠다") — 관계가 거래 단위로 축적(아크 제목 "기브 앤 테이크"가 문법 선언). → 우리 relation·동료 시스템의 서사 문법.
- **NPC의 판단 문법**: 베테랑은 묻기 전에 관찰로 결론("초심자군") — NPC마다 정보 처리 방식이 성격.
- 주인공 냉소 내레이션 ↔ 진심 순간의 대비(감정 낙차 장치). 극한 상황 = 자아 분열 내적 대화(광기 연출을 대화 형식으로).

### 축 3 — 밸런스·성장의 재미
- **선택지에 3층 근거**: 방패 선택 이유 = ①경제(되팔이 최고가) ②현실 역량(날붙이 못 씀) ③최종 빌드(방패바바) — 좋은 선택지는 경제·역량·정체성이 한꺼번에 걸림. → 조우 선택지 저작 규칙 후보.
- **보상 곡선은 바닥부터**: 첫 전투 = 다리 하나 걸고 이긴 대가가 "빵 한 조각"(9등급 마석) — 초반 보상의 처참함이 이후 성장의 체감을 만듦. 화폐 = 마석 등급제(9등급~).
- **위기 자체가 보상**: 「업적 달성 — 생명력 2% 이하 / 정신 +1 영구」 — **고난이 영구 성장으로 정산되는 업적 설계 = 우리 achievements/perk(실패도 남긴다)와 정확히 동형.** 검증된 재미 문법.
- **통합 전투력 지표**: 「종합 전투 지수 67(+6)」 — 세부 스탯 위에 단일 체감 지표 1개(성장이 한 숫자로 읽힘). + 「출혈 지속」 반복 로그 = 서스펜스 메트로놈(같은 라인의 반복이 긴박 리듬).
- 지식이 스탯을 이김: 초반 전투력 = 육체 스탯이 아니라 게임 지식×아이디어×대가 지불 — "아는 자의 우위"가 육성 전 단계의 재미(우리 조사 레인·명명 단서의 재미 근거).

## 4. 스테이지드 학습 계획 (후속 세션 — 전권 정독은 분할)

1. 1~30화 정독: 초반 성장 곡선·시스템 도입 순서(스탯·스킬·동료 획득 페이스) — 우리 육성 페이싱의 겜판 기준.
2. 아크 경계 20곳 정독: 아크 개폐 문법(문제 제기 화 vs 정산 화 구조).
3. 동료(NPC) 관계 아크 추적: 겜판에서 NPC가 인격을 얻는 화차·계기 — 우리 동료 시스템·후일 AI 상호작용의 서사 데이터.
4. 떡밥 회수 간격 실측: 던진 정보(5형)가 회수되는 화 거리 분포 — 콜백 저작 규칙의 정량 근거.
