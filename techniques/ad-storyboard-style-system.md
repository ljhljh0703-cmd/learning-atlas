# 광고 SB 덱 스타일 시스템 — 무채색 에디토리얼

> **무엇**: 작가가 최종 승인한 광고 스토리보드 덱(`SB_Ver.01.pptx`)에서 역추출한 *시각 계약*. 앞으로 모든 SB/구성안 덱은 이 시스템을 기본값으로 따른다. (HARNESS AS-09 clean-room·AS-11 비율불변·AS-08/12 광고법의 *표현층 구체 인스턴스*.)
> **관계**: HARNESS = 작업 프로토콜(규율), 본 노드 = *덱 비주얼 정본*(토큰·레이아웃). SKILL M2 표현층이 이 시스템으로 산출.

## 한 줄 요약
**무채색 미니멀 에디토리얼** — 흰 배경 · 굵은 좌측정렬 제목 · 검정 칩/「」자막 · 사진은 풀블리드 16:9. 색은 오직 *사진(제품)* 에서만.

## 슬라이드 흐름 (기본 골격)
1. **외주처 표지** — 제작사 템플릿 그대로 둠(작가가 만들지 않음). 본 작업은 2장부터.
2. **타이틀** — 흰 배경, 큰 검정 볼드 제목 + 상단 eyebrow(`STORYBOARD · 시안`, 자간 넓게) + 우측 제품컷 + 하단 얇은 메타 한 줄.
3. **영상 개요** — 표(분류 | 내용).
4. **컨셉 썸네일** — A/B 카드: 검정 칩(A/B) + 타겟 + 원메시지「」 + 큰 이미지에 **자막 번인**(굵은 흰 글자 + 검정 하이라이트).
5. **A 컷 구성** — 표(컷·시간·화면 설명·대사/내레이션·카피).
6. **A 6컷 이미지 그리드** (2×3).
7. **B 컷 구성** — 표.
8. **B 6컷 이미지 그리드** (2×3).

## 컬러 토큰
| 용도 | 값 |
|---|---|
| 배경 | `#FFFFFF` |
| 제목·본문 | `#1A1A1A` (니어블랙) |
| 보조 텍스트 | `#5C5C5C` |
| 칩·표 헤더 | `#222222` (차콜) |
| 라인/보더 | `#DBDBDB` (헤어라인) |
| 표 교차행 | `#FFFFFF` / `#F5F5F5` |
| 강조색 | **없음** — 색은 사진(제품)에서만. 무채색 원칙 고수. |

## 타이포그래피
- 한글 고딕(맑은 고딕 계열), 제목 **헤비 웨이트**, 전부 **좌측 정렬**.
- **eyebrow 라벨**: 대문자/자간 넓게 (`S T O R Y B O A R D · 시안`).
- **섹션 제목 아래 짧은 언더라인 바**(제목 폭의 1/4 정도)가 시그니처.
- 본문은 담백·간결. 과한 볼드 남발 X.

## 컴포넌트 규칙
- **칩(badge)**: 검정 사각 박스 + 흰 볼드 글자. A/B 라벨, 컷 번호(01·02…)에 사용.
- **자막 표기**: 코너 브래킷 `「 」` + 볼드. (효능 단어 X — 분위기·관계·행동 카피.)
- **시간 표기**: 컷 우상단 흰 박스 pill (`0–5s`).
- **메타 행**: 하단 한 줄, `·` 구분, 작은 글자 (클라이언트 · 규격 · A · B).
- **컨셉 썸네일 자막 번인**: 히어로 이미지 위 큰 볼드 흰 글자 + 검정 하이라이트(유튜브 썸네일 느낌).
- **표**: 차콜 헤더(흰 볼드) + 흰/연회색 교차행 + 얇은 회색 라인. 화면·대사 **좌측 정렬**, 컷·시간·카피 **가운데**, 카피 볼드.

## 이미지 규칙
- 16:9 **원본 비율 유지**(크롭·왜곡 금지 — 프레임을 16:9로). 컷 그리드는 2×3. → AS-11 비율불변.
- 제품 팩샷은 실제 제품, 로고 실물.

## 광고법 (불변)
- 효능 단정(집중력·기억력·뇌·각성·피로회복 등) 금지. 자막/카피는 상황·맛·관계만. → AS-08/AS-12 표현층 적용.

## 하지 말 것
- 임의 컬러 팔레트·테마색 입히기 (무채색 깨지 말 것).
- 페이지 순서 임의 변경/추가 (지정 순서 고수).
- 사진 강제 크롭으로 인물·제품 잘림.

## 적용
- 신규 광고 SB 착수 시 SKILL M2 표현층의 *기본 덱 시스템*으로 로드. 클라이언트 브랜드 가이드가 별도 톤을 요구하면(AS-05 브랜드 연속성) 그때만 분기.
- 이 시스템에서 일탈하는 새 스타일 요청이 *반복*되면 → 본 노드에 variant 추가 또는 HARNESS 정련(M3 학습 엔진).
