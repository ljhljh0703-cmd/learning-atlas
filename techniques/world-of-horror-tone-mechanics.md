# WORLD OF HORROR — 톤·메커니즘 레퍼런스

> ⚠️ 임시(provisional). 작가 지정 톤 레퍼런스 3종 중 하나(엘드리치 호러·아컴 호러·**WoH**). 용도 = eldritch-seoul-rpg-tone-bible 입력.

## 1. 정체 (1줄)
1980년대 일본 항구도시 시오카와, 이토 준지+러브크래프트 감성의 **1-bit 그림판 도트** 턴제 코스믹 호러 로그라이트(panstasz 1인 개발, 2023 정식).

## 2. 메커니즘 (엘드리치 서울 관련분만)
- **파멸(DOOM) = 전역 %트랙**: 조사·휴식·이벤트가 조금씩 올리고, 100% = 고대 신 강림 = 게임오버. 파멸을 *깎는* 수단은 희소·유료(동료 희생, 자금). → 우리 causality(8→0)와 동형(방향만 반대).
- **고대 신 = run 변형자(modifier)이지 보스가 아님**: 선택/무작위로 1주 배정 → 각자 **특규**(전투 도주 불가·휴식 페널티·매턴 금지구역 명령·이성↓=파멸↑ 등) + 전용 이벤트 + 전용 엔딩 문구. 직접 전투 불가 — *싸울 수 없는 최종 보스 = 페널티 시스템*. → 시나리오별 "괴이"를 run 변형자로 설계하는 문법(신화 풀에 시나리오 괴이 색 입히기, D2와 직결).
- **미스터리 = 도시괴담 앤솔로지**: run마다 미스터리 여러 개(각 2~4 엔딩), 학교·병원·바닷가·등대 등 일상 지명 순회. → 시나리오 카탈로그(B3)의 검증된 원형 — 단편 괴담 묶음 + 공유 시스템.
- **동료 = 보너스 + 소모 가능 자원**: 스탯/슬롯 보너스 + 전투 주의끌기(실패 시 사망) + "적에게 보내기"(아이템처럼 소모) + **고이 보내줄 때만 특전**. 동료마다 한 줄 대사 `<<...>>`. → 우리 D12 absorb·이탈 설계와 동형 + *희생의 선택지화*(잔혹 경제)가 추가 재미 축.
- **해금 루프**: 엔딩/업적 → 추가 캐릭터·숨겨진 장소·고대 신 해금. 수치 부스트 아닌 **콘텐츠 언락** — 우리 언락 위주 메타(A3)의 실증 선례.
- 저주/부상 = 지속 디버프 카드(청구서 계열 친척).

## 3. 톤 해부 (톤 바이블 입력 — 핵심)
1. **보도체 괴담**: 시놉·이벤트가 신문 보도/증언 문체 — "실종자는 늘어만 간다", "역겨운 괴생명체들이 해안가를 공포에 빠트리고 있다". 감정 형용 대신 **사실 나열 + 말줄임(...)의 여운**.
2. **일상의 침식**: 공포의 무대 = 학교·병원·전차·등대 — 낯선 곳이 아니라 **아는 곳이 어긋남**. 이토 준지 문법.
3. **극단적 절제**: 1-bit 그래픽과 등가인 텍스트 미니멀리즘 — 짧은 단락, 수식어 최소, 잔혹은 **한 줄 요약**으로("장기가 크탁-아토라스의 독에 녹아내렸다").
4. **불길한 예고형 flavor**: 고대 신 소개 1~2문("거울 속 당신의 얼굴 뒤로 언뜻 보이는 광경은 진짜 악몽의 시작일 뿐") — 위협 배너 문형의 원형.
5. **동료 한 줄 인격화**: `<<괜찮을 거야, 날 믿어>>` — 최소 비용으로 애착 형성 → 희생의 무게.
6. 2인칭 지시("당신은 마침내 피할 수 없는 그 마을에 발을 딛는다") — 우리 기생충 나래이터 GM식 2인칭과 정합.
